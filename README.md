## Design Principles

- Single Responsibility Principle (SRP) (单一功能原则)
- Interface Segregation Principle (ISP) (接口分离原则)
- Liskov Substitution Principle (LSP) (里氏代换原则)
- Open-Closed Principle (开闭原则)
- Dependency Inversion (依赖反转原则)
